# oneplayer

## Setting up
[Install yarn](https://yarnpkg.com/en/docs/install)

Install typescript
```bash
yarn global add typescript
```

### libSAMP
```bash
cd libsamp
```

Install node
```bash
yarn add node
```
Install
```bash
yarn install
```
