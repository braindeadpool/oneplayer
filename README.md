# oneplayer

## Setting up
Install nvm as described [here](https://github.com/nvm-sh/nvm#install--update-script)
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.0/install.sh | bash
```
Install node
```bash
nvm install node
```
Install typescript
```bash
npm install -g typescript
```
